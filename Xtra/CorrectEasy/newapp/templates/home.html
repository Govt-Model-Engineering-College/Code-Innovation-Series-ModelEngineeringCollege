Welcome to CorrectEasy

CorrectEasy is an Django appplication which uses Google's Vision API to analyze images.
CorrectEasy makes it easier for teachers to calculate the marks of each question . The teacher simply needs to upload the key words of the answer along with the image of the students answer. We will evaluate the answer and predict an approximate marks that could be given to the students.

Steps to use CorrectEasy

  • Upload student's answer key
  • Enter answer key
  • CorrectEasy predicts the marks

Upload student's answer

CorrectEasy makes it easier for teachers to calculate the marks of each question. In the first step, the teacher simply needs to upload the answer sheet of the students.

Enter answer key

Once the teacher uploads the student answer sheet, next enter the answer key to identify the key points in the answer.

Mark prediction

CorrectEasy helps evaluate the answer and predict the approximate mark that can be given to the students. With the help of Google's Vision API, the handwritten texts can be quickly identified and helps get the results faster

Features available in CorrectEasy

Label Detection

Provides generalized labels for an image. Returns a textual description, confidence score, and topicality rating for each label. Confidence level is the level of confidence by the detection algorithm that the image contains a particular label.

Text Recognition

Optical character recognition for an image, text recognition and conversion to machine-coded text. Identifies and extracts UTF-8 text in an image. Works with currently supported languages.

Handwriting Recognition

CorrectEasy detects handwritten text. Optimized for dense areas of text in images that contain handwriting.Works with different languages and gives extremely fast and highly accurate results.

Crop Hint Recognition

Provide upto 16 image ratio values for an image.

Image properties

Returns dominant colors in an image. Each color is represented in the RGBA color space, has a confidence score, and displays the fraction of pixels occupied by the color [0, 1].

Dense Text Recognition

Optical character recognition (OCR) for a file (PDF/TIFF) or dense text image; dense text recognition and conversion to machine-coded text.

Math equation recognition

Detects math equations and formulae.Take images of your calculations, equations, chemical formulae and get instant results.
